Client Portal Web API is based on Interactive Brokers' Client Portal.

You can place trades, request market data, retrieve account and portfolio information, etc.

Client Portal Web API is not capable of creating and managing accounts (For that, we offer DAM API)

You can start by following Getting Started section or review existing Endpoints
